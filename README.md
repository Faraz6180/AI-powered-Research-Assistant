Multiple platforms (ArXiv, Wikipedia, web)

No single intelligent interface
💡 Solution

This app:

Retrieves data from multiple sources in parallel

Provides instant answers in chat format
🏆 Key Features
🔍 Multi-source search (ArXiv + Wikipedia + Web)
🤖 AI-powered synthesis using Groq
💬 Conversational interface with memory
⚡ Fast results (parallel processing)
🎨 Clean UI built with Streamlit
🛠️ Tech Stack
Frontend: Streamlit
LLM: Llama 3.3 70B via Groq
Framework: LangChain
APIs: ArXiv, Wikipedia, DuckDuckGo
Language: Python
